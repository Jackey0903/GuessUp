# 吾猜 (GuessUp)

吾猜（GuessUp）是一款 NBA 球员竞猜微信小程序。
游戏设计主打“氛围感”：暗黑模式、霓虹橙色调以及流畅的交互动画。

## 🚀 快速开始

### 准备工作
1.  **微信开发者工具** (WeChat Developer Tools)。
2.  **AppID**: 你可以使用自己的 AppID，或者使用测试号。

### 安装步骤
1.  克隆或下载本项目代码。
2.  打开 **微信开发者工具**。
3.  选择 **导入项目** (Import Project)。
4.  选择 `GuessUp` 文件夹。
5.  点击 **编译** (Compile) 运行预览。

---

## 🏀 玩法说明
1.  **开始**: 点击主页的 "START GAME" 按钮。
2.  **猜测**: 在搜索框输入球员名字（支持中文如 "詹姆斯" 或英文 "LeBron"）。

### 🎮 游戏规则 (Game Rules)

每一天，你需要猜出一名 NBA 现役球员（2024-2025 赛季）。你有 **8 次** 竞猜机会。

每次猜测后，表格会通过颜色给出提示：

*   🟩 **绿色 (Green)**: **完全匹配！** (属性完全一致)
*   🟨 **黄色 (Yellow)**: **部分匹配！** (接近了)
    *   **位置 (Pos)**: 虽不完全相同，但包含共同位置 (如: 猜后卫 PG，实际是双能卫 PG/SG)。
    *   **身高 (Height)**: 误差在 **5cm** 以内。
    *   **年龄 (Age)**: 误差在 **2岁** 以内。
    *   **号码 (#)**: 误差在 **2号** 以内。
*   ⬛ **灰色 (Gray)**: **完全不匹配**。
*   ⬆️ ⬇️ **箭头**: 提示目标球员的属性数值是更高还是更低。

如果 8 次都没猜对，游戏结束！🔥即可获胜！

## 🛠 数据管理

### 球员数据
游戏默认加载 `utils/players.js` 中的数据。
*   **完整数据**: 包含 **25-26 赛季** 500+ 名现役球员（自动抓取自 ESPN）。
*   **中文支持**: 热门球星（约 50 名）支持中文别名搜索（如 "库里", "老北京"）。其他球员暂时仅支持英文搜索。
*   **数据字段**: 包含身高 (cm)、位置 (中文)、球队 (中文)、选秀年份等。

### 更新数据 (可选)
如果你想获取最新的 NBA 英文名单：
1.  运行 Python 脚本：
    ```bash
    python3 scripts/fetch_nba_data.py
    ```
2.  脚本会生成 `utils/players_full.js`。
3.  若要使用完整名单，请将其重命名为 `utils/players.js`（**注意**：这样做会覆盖掉手动整理的中文别名，导致无法用中文搜索）。

## 📂 项目结构
*   `pages/`: 页面文件 (Home, Game)。
*   `utils/`: 工具函数和数据。
*   `scripts/`: 自动化脚本 (数据抓取)。
